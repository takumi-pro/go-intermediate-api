## 3. データベースの扱い方

- データベース接続には`database/sql`パッケージを使用する
- 実際にデータベースと通信しているのは`database/sql`ではなくドライバ
- postgresでは`github.com/lib/pq`を使用する
- `database/sql`はデータベースへの接続を抽象化して異なるデータベースでも同じメソッドで接続やデータ取得・疎通確認ができる（ポリモーフィズム?）
